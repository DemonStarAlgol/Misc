且额外数据均为100字节长的字符串，且疑似base64编码。尝试按文件名顺序依次将压缩包解压得到图片，并提取图片末尾的字符串后拼接：
import zipfile
path1 = 'C:/Users/Administrator/Desktop/flag/'
path2 = 'C:/Users/Administrator/Desktop/flag/out/'

res = ''
for i in range(1, 87):
    zip = zipfile.ZipFile(path1 + str(i) + '.zip')
    jpg = str(i) + '.jpg'
    zip.extract(jpg, path2)
    zip.close()

    f = open(path2 + jpg, 'rb').read()
    res += f[len(f)-100:len(f)+1].decode('utf-8')
print(res)

通过CRC32爆破的方法得到9个txt文件的内容：
import zipfile
import string
import binascii

crc = []
zip = zipfile.ZipFile('C:/Users/Administrator/Desktop/flag.zip', 'r')
for i in zip.namelist():
    crc += [zip.getinfo(i).CRC]
crc = crc[1:10]

for i in range(9):
    for j in string.printable:
        c = binascii.crc32(j.encode('utf-8'))
        if c == crc[i]:
            print(j, end = '')

f = open('C:/Users/Administrator/Desktop/n.png', 'rb').read()
