018 再见李华
题目：假如你是李华（LiHua)，收到乔帮主一封密信，没有任何特殊字符，请输入密码，不少于1000个字。同学，记得署名哦。

十六进制编辑器打开，末尾有zip压缩包，提取之。打开是个带密码的key.txt，且不是伪加密。那么需要密码爆破。
根据题目提示，密码不少于1000（即十进制的8）个字母，且最后应为LiHua。进行MD5爆破。

8位的情况：
from hashlib import md5
import string

dic = string.printable
a = '1a4fb3fb5ee12307'

for i1 in dic:
    for i2 in dic:
        for i3 in dic:
            str = i1 + i2 + i3 + 'LiHua'
            str_md5 = md5(str.encode("utf8")).hexdigest()
            if a in str_md5:
                print(str)
                exit(0)
跑完了什么都没有，增加到9位：
from hashlib import md5
import string

dic = string.printable
a = '1a4fb3fb5ee12307'

for i1 in dic:
    for i2 in dic:
        for i3 in dic:
            for i4 in dic:
                str = i1 + i2 + i3 + i4 + 'LiHua'
                str_md5 = md5(str.encode("utf8")).hexdigest()
                if a in str_md5:
                    print(str)
                    exit(0)
结果为15CCLiHua，解压得到flag。

flag：Stay hungry, Stay foolish.
