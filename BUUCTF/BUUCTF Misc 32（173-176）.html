173 静静听这么好听的歌
题目：小明喜欢听歌，更喜欢收集音乐。一天小明收集到了一份据说是从上世纪七十年代保存下来的音频文件，该文件被认为是某次特殊行动的秘密文件。然而令人费解的是这么多年了它一直是破损的无法正常打开，而随音乐附带的一段神秘歌词更是让音乐界百思不得其解。曲子作者是一个叫L.S.B的上校，据称此人的一切痕迹都被抹去了，所以查不到任何信息。现在已经2015年了，你是否能找出四十多年前的秘密？

下载得到wav格式文件和LSB.txt。后者内容：

这是一段MATLAB脚本，大致上是将一张bmp格式的二值图像（只有黑白两种像素）通过低位水印方式写入了wav格式音频中。引用一下大佬的Writeup：
clc;
clear;

marklength=38800;
row=388;                   
fid=fopen('静静听这么好听的歌.wav','r');
oa=fread(fid,inf,'uint8');
status=fseek(fid,44,'bof');
a=fread(fid,marklength,'uint8');
for i=1:marklength
    w1(i)=bitget(a(i),1);
end
w1=w1';
m=Vector2Matrix(w1,row);
imwrite(m,'flag.bmp','bmp');
figure;
imshow('flag.bmp');title('extracted watermark');
%其中Vector2Matrix函数：
function A=Vector2Matrix(oi,row)
%将数组转换成二位数组,row为行数
cl=length(oi)/row;
A=zeros(row,cl);
for i=1:row
    for j=1:cl
        A(i,j)=oi(cl*(i-1)+j);
    end
end
由于从wav文件中提取得到的信息是一维的，损失了原始图片的尺寸信息，需要自己爆破得到宽度或高度。宽度为100时可以得到正确的图片。

174 [网鼎杯 2020 青龙组]虚幻2

下载得到file文件，十六进制编辑器查看文件头，发现是png格式。重命名得到36x12大小的png图像。
放大后观察发现，图中共有八种颜色：黑、白、红、绿、蓝、黄、紫、天蓝。用Gimp取色发现，所有颜色都是纯色，即颜色的RGB值只有255和0两种。
尝试把每个像素的RGB值提取之后，将255和0转成1和0的二值数据。由于原图的宽36刚好是高12的3倍，二值数据共有36x36位，刚好排成一个正方形。猜测原图每个像素的RGB值，对应36x36方阵中纵向排列的3个二进制值。

首先按列优先提取原图像素的RGB数据，255对应1，0对应0：
from PIL import Image

res = ''
img = Image.open('C:/Users/Administrator/Desktop/file.png')
for i in range(36 * 12):
    x = i // 12
    y = i % 12
    p = img.getpixel((x, y))
    res += str(p[0] // 255)
    res += str(p[1] // 255)
    res += str(p[2] // 255)
然后按1为白，0为黑，仍然列优先，转为36x36大小的二值图像：
pic = Image.new('RGB', (36, 36))
for i in range(len(res)):
    x = i // 36
    y = i % 36
    if res[i] == '1':
        pic.putpixel((x, y), (255, 255, 255))
    elif res[i] == '0':
        pic.putpixel((x, y), (0, 0, 0))
pic.show()
